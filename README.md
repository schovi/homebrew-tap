# Homebrew Tap

Homebrew formulae for schovi's tools.

## Installation

```bash
brew install schovi/tap/shelli
```

Or:

```bash
brew tap schovi/tap
brew install shelli
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [shelli](https://github.com/schovi/shelli) | Persistent interactive shell sessions via PTY |
